    • Indie Game Bash

      The Indie Game Bash is a celebration of independent couch multiplayer video games in Calgary, Alberta. The event showcased acclaimed indie games from the Canada, US and Netherlands, as well as Canadian and local Calgary games. A cocktail bar and food truck were setup to provide food and refreshments.

    • SAIT 2D Game Design Curriculum

      A curriculum for the Southern Alberta Institute of Technology which teaches teens how to make a 2D game using GameMaker: Studio. The goal for the camp was to have the students work on a game by building objects and giving them behaviours using the Drag'N'Drop system within GameMaker: Studio. The 2015 Summer session was completely sold out.
